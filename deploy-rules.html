Firebase Rules Deployment

Instructions:

  1. Go to the Firebase Console Rules Page
  2. Replace the current rules with the rules below
  3. Click "Publish" to deploy the rules
  4. Return to this page and click "Continue to Setup"

Rules to Copy:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // TEMPORARY: Allow all operations during setup
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
